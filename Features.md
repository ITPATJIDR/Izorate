Sessions (ปัจจุบันทำอยู่): หน้าจัดการการเชื่อมต่อ, Server, Group และใช้งาน Terminal หลัก
Files: หน้าจัดการไฟล์ (SFTP/FTP) แบบลากวาง (Drag & Drop) เพื่อโอนย้ายไฟล์ข้าม Server โดยไม่ต้องพิมพ์คำสั่ง scp
Tools: แหล่งเครื่องมือ Network สำหรับแอดมินหรือ Developer เช่น Port Scanner, Ping/Trace Route Tool หรือ Network Calculator รวมอยู่ในแอปเดียว
Keys: ระบบจัดการ SSH Keys (Public/Private Keys) เพื่อเพิ่มความปลอดภัย ให้เราสร้าง, นำเข้า หรือ Assign กุญแจให้แต่ละ Server ได้ง่ายๆ โดยไม่ต้องคอยจำรหัสผ่าน
AI Assistant: หน้าต่างผู้ช่วย AI เต็มรูปแบบที่เชื่อมกับ Terminal ช่วยวิเคราะห์ Error หรือเขียน Shell Script/Docker Compose ให้เราได้ทันที (ตอนนี้มี Component AIPanel เตรียมไว้แล้วบางส่วนทางขวา)
